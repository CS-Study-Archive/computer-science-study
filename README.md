## ✅ GIT CONVENTION

### 1) COMMIT

- `type: 자유 `
- **type: 하고자 하는 행위를  명령문으로 적는다.** 
  - Add  Files
  - Update Files
  - Delete Files
  - Merge Files
  - etc..

```bash
git commit -m "Add Files: 발표자료 업로드 "
```

<br>

### 2) PR 

- PR 제목: `이름 / 회차 / 큰 주제 / 세부 주제`

```
박재경 / 1회차 / 운영체제 / 프로세스와 스레드
```

<br>

- PR 내용: 자유 
  - 다른 스터디원들이 자료를 이해하는 데 도움이 되는 설명을 간단히 적어주세요.
  - 발표자료를 만들면서 느낀 간단회고를 작성하는 것도 좋습니다.
